# Store Listing Texts

## Google Play Store

### Title
YouTubeWhitelist - Safe YouTube for Kids

### Short Description (80 chars max)
Whitelist-based YouTube client for kids. Parents approve, kids watch safely.

### Full Description

YouTubeWhitelist is a free, open-source Android app that gives parents full control over what YouTube content their kids can watch.

HOW IT WORKS

Parents browse YouTube and whitelist specific channels, videos, and playlists. Kids only see approved content — nothing else. No algorithm, no recommendations, no surprises.

FEATURES

- Parent Mode: Browse YouTube freely and build your kid's whitelist
- Kid Mode: Clean, distraction-free interface with only approved content
- PIN Protection: Secure parent mode with a PIN code
- Multiple Profiles: Create separate whitelists for each child
- Daily Time Limits: Set per-profile daily watch time limits
- Sleep Mode: Timer-based playback with gradual volume fade-out for bedtime
- Watch Statistics: Track daily, weekly, and monthly watch time
- Kiosk Mode: Screen pinning keeps kids inside the app
- Export / Import: Backup and restore profiles and whitelists
- Search: Kids can search within their whitelisted content
- Playlist Support: Whitelist entire playlists

PRIVACY & SECURITY

- 100% client-side — no backend server, your data stays on your device
- No ads, no tracking, no data collection
- Open source (GPLv3) — inspect the code yourself
