# YouTubeWhitelist
A YouTubeWhitelist egy ingyenes, nyílt forráskódú Android alkalmazás, amely lehetővé teszi a szülők számára, hogy whitelist alapún szabályozzák, gyermekük milyen YouTube tartalmakat nézhet. Az alkalmazás két módban működik: szülői módban (teljes YouTube böngészés és whitelist kezelés) és gyerek módban (kizárólag a whitelistelt tartalmak elérhetők).
